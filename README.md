# Slider Interactivo

Proyecto de práctica personal desarrollado con **HTML y CSS**.

**Descripción:**  
Un slider interactivo donde los usuarios pueden desplazarse por el contenido usando `Shift + rueda del ratón` en desktop o swipe en dispositivos móviles.  

**Aprendizaje y buenas prácticas aplicadas:**  
- **Flexbox** y **estructuras de layout** para organizar los elementos del slider de manera responsiva.  
- **Metodología BEM** para nombrar clases de forma clara y consistente (`slider`, `slider__item`, `slider__item--active`).  
- Aplicación de **buenas prácticas de código** para mantener la legibilidad y escalabilidad del proyecto.

**Tecnologías:** HTML, CSS
